# AI Learning Hub Web App

This is a static HTML web application generated from your notes.

## How to Run

1.  Open the `index.html` file in your web browser.
    *   You can right-click `index.html` in VS Code and select "Open with Live Server" if you have the extension installed.
    *   Or, simply double-click the file in your file explorer.

## Structure

*   `index.html`: The main landing page.
*   `topics/`: Contains detailed HTML pages for each topic.
    *   `search.html`: Contains the **Interactive Animations** for BFS, DFS, and the Water Jug Problem.
*   `css/`: Stylesheets.
*   `js/`: JavaScript files for animations.
