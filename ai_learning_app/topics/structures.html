4. Knowledge Representation Structures

A. Weak Slot-and-Filler Structures

Organize knowledge as objects with attributes (slots) and values (fillers).

1. Semantic Networks

A graph where nodes are concepts and arcs are relationships.

  • ISA: Subclass relationship (Bird ISA Animal).
  • INSTANCE: Specific object (Tweety INSTANCE Bird).
  • HASPART: Composition (Bird HASPART Wings).

2. Frames

Data structures representing a stereotypical situation. Like a class in OOP.

Frame: Bird
  ISA: Animal
  Wings: 2
  Legs: 2
  Fly: True

B. Strong Slot-and-Filler Structures

Specific to representing events and natural language meanings.

1. Conceptual Dependency (CD)

Represents the meaning of sentences independent of the specific language. Uses primitive actions (ATRANS, PTRANS, MTRANS, etc.).

2. Scripts

Describes a stereotyped sequence of events in a particular context (e.g., A Restaurant Script).

  • Entry Conditions: Hungry, have money.
  • Scenes: Entering, Ordering, Eating, Paying.
  • Results: Not hungry, less money.
