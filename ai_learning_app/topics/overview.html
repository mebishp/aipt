1. Overview of Artificial Intelligence

A. Definition and Goals

Artificial Intelligence (AI) is the simulation of human intelligence processes by machines. It aims to create systems that can function autonomously.

Key Goals

  • Learning: Acquiring information and rules for using it.
  • Reasoning: Using rules to reach approximate or definite conclusions.
  • Problem Solving: Identifying solutions to complex issues.
  • Perception: Interpreting sensory data (e.g., vision, speech).

Why do we need AI?

Reduce Errors

Machines don't get tired or distracted.

24x7 Availability

AI agents can work continuously without breaks.

Unbiased Decisions

AI follows logic, reducing emotional bias.

B. Types of AI Systems

Type Description Examples
Narrow AI (Weak) Designed for a specific task. Siri, Alexa, Netflix Recommendations.
General AI (Strong) Can perform any intellectual task a human can. Theoretical (Sci-Fi robots).
Superintelligence Surpasses human intelligence in all aspects. Speculative.

C. Intelligent Agents

An agent is anything that can be viewed as perceiving its environment through sensors and acting upon that environment through actuators.

The PEAS Framework

To design a rational agent, we must define its task environment:

  • Performance Measure: How do we define success?
  • Environment: Where does the agent operate?
  • Actuators: How does it affect the world?
  • Sensors: How does it perceive the world?

Example: Autonomous Taxi

Performance: Safety, speed, comfort, profit.

Environment: Roads, traffic, pedestrians.

Actuators: Steering, accelerator, brake.

Sensors: Cameras, GPS, speedometer.

D. Types of Environments

The environment determines the design of the agent.

  • Fully vs. Partially Observable: Can the agent see the whole state? (Chess vs. Poker)
  • Deterministic vs. Stochastic: Is the next state completely determined by the current state and action? (Puzzle vs. Driving)
  • Static vs. Dynamic: Does the world change while the agent is thinking? (Crossword vs. Taxi)
  • Discrete vs. Continuous: Are there limited states? (Chess vs. Driving)

E. Agent Types by Complexity

1. Simple Reflex Agent

The most basic agent. Works on condition-action rules directly.

  • Perceives environment → Applies rule → Performs action.
  • Limitation: Only works in fully observable environments.
  • Example: A vacuum cleaner that cleans whenever dirt is detected.

2. Model-Based Reflex Agent

Maintains an internal model of the world to handle partial observability.

  • Keeps track of past percepts (internal state).
  • Uses a model: "How does the world change?"
  • Example: A robot that remembers where walls are even when not currently visible.

3. Goal-Based Agent

Extends model-based agent by having explicit goals.

  • Searches for a sequence of actions to achieve the goal.
  • Enables planning and lookahead.
  • Example: GPS navigation (goal: reach destination).

4. Utility-Based Agent

Similar to goal-based but measures success on a continuous scale.

  • Uses a Utility Function to rank states by preference.
  • Selects actions that maximize expected utility.
  • Example: A taxi that optimizes for speed, comfort, AND profit.

5. Learning Agent

Can improve its performance over time through experience.

  • Learning Element: Modifies the agent's behavior based on feedback.
  • Performance Element: Selects external actions.
  • Critic: Provides feedback against a standard.
  • Example: Machine learning models that improve with training data.

F. Behavioral Characteristics

Intelligent agents typically exhibit these properties:

  • Reactivity: Quick response to environment changes.
  • Proactiveness: Taking initiative to achieve goals (not just reacting).
  • Autonomy: Operating independently without constant human control.
  • Adaptability: Learning and improving from experience.
  • Goal-Oriented Behavior: Actions directed at specific objectives.
  • Persistence: Continuing despite obstacles or failures.
  • Efficiency: Achieving goals with minimal resource usage.
  • Collaboration: Working with other agents or humans.

G. Applications of AI Agents

Robotics

Autonomous robots for manufacturing, exploration, and assistance.

Gaming

AI opponents in video games (chess engines, game bots).

Virtual Assistants

Siri, Alexa, Google Assistant—perceive voice and act on commands.

Autonomous Vehicles

Self-driving cars that perceive the road and navigate safely.
