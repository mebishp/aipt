7. Constraint Satisfaction Problems (CSP)

A. Definition

A CSP consists of:

  • Variables: $X = \{X_1, \dots, X_n\}$
  • Domains: $D = \{D_1, \dots, D_n\}$ (Possible values for variables)
  • Constraints: $C$ (Rules limiting values)

Goal: Assign a value to every variable such that all constraints are satisfied.

B. Examples

  • Map Coloring: Color a map so no two adjacent regions have the same color.
  • N-Queens: Place N queens on an NxN board so none attack each other.
  • Sudoku: Fill grid so every row, column, and box has digits 1-9.

C. Backtracking Search

The primary method for solving CSPs.

It is a Depth-First Search that assigns values to variables one by one.

  1. Select an unassigned variable.
  2. Try a value from its domain.
  3. Check constraints. If valid, move to next variable.
  4. If invalid, try next value.
  5. If no values work, Backtrack to previous variable.
