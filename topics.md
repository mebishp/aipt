# AI Learning Hub - Core Topics (Based on Course Notes)

## Topics 1-7: Foundation Curriculum

1.  **Overview of Artificial Intelligence and Agents**
    *   Definition and Goals of AI
    *   Types of AI Systems (Narrow, General, Superintelligence)
    *   Intelligent Agents and Rationality (PEAS Framework)
    *   Agent Types (Simple Reflex, Model-based, Goal-based, Utility, Learning)
    *   Behavioral Characteristics (Reactivity, Autonomy, Adaptability)
    *   Types of Environments (Observable, Deterministic, Episodic, Discrete, Multi-Agent)

2.  **Problem Solving and Search Techniques**
    *   State Space Search and Problem Formulation
    *   Problem Characteristics (Decomposability, Reversibility, Predictability, Solution Type)
    *   Uninformed Search (BFS, DFS, UCS, Bidirectional Search)
    *   Informed Search (Greedy Best-First, A*, Hill Climbing, Generate-and-Test)
    *   Search Algorithm Comparison (Completeness, Optimality, Time/Space Complexity)
    *   Interactive Animations (BFS/DFS visualization, Water Jug Problem solver)

3.  **Knowledge Representation and Logic**
    *   KR Fundamentals (Facts, Representations, Mappings)
    *   Logical Schemes (Propositional Logic, Predicate Logic with Quantifiers)
    *   Inference Rules (Forward Chaining, Backward Chaining with detailed comparison)
    *   Resolution in Predicate Logic (Proof by Contradiction in 5 steps)
    *   Types of Reasoning (Deductive, Inductive, Abductive)

4.  **Knowledge Representation Structures**
    *   Weak Slot-and-Filler Structures
        - Semantic Networks (ISA, INSTANCE, HASPART relations)
        - Frames (Classes, Instances, Inheritance, Attributes)
    *   Strong Slot-and-Filler Structures
        - Conceptual Dependency (CD for representing events)
        - Scripts (Stereotyped sequences with Entry Conditions, Roles, Scenes, Results)
        - CYC (Common Sense Knowledge Base)

5.  **Planning**
    *   Planning vs. Search (Differences and Representations)
    *   Planning Components (Initial State, Goal, Operators with Preconditions/Effects)
    *   STRIPS (Goal Stack Planning)
        - Push-down Stack, State List management
        - ADD list, DELETE list, Pre-Condition list
        - Algorithm steps and execution order
    *   TWEAK (Non-Linear Planning with Constraint Posting)
    *   Handling Goal Interactions (Sussman Anomaly)

6.  **Learning in AI**
    *   Classification of Learning Models (Supervised, Unsupervised, Reinforcement, Semi-Supervised)
    *   Learning Mechanisms
        - Rote Learning (Memorization/Caching)
        - Learning from Examples (Induction)
        - Explanation-Based Learning (EBL) - generalize from single example
        - Learning in Problem Solving (Parameter Adjustment, Chunking)
    *   Decision Trees (ID3 Algorithm with Entropy and Information Gain)
    *   Reinforcement Learning (Agent-Environment, Q-Learning, Policy Gradient, Actor-Critic)
    *   Neural Networks (Perceptron, MLP, CNN, RNN, Training via Backpropagation)
    *   Genetic Algorithms (Population, Fitness, Selection, Crossover, Mutation)
    *   Learning Approaches Comparison Table

7.  **Constraint Satisfaction Problems (CSP)**
    *   CSP Definition (Variables, Domains, Constraints)
    *   Problem Examples (N-Queens, Map Coloring, Sudoku)
    *   Backtracking Search Algorithm
    *   Constraint Propagation Techniques

## Learning Features

*   **Interactive Visualizations:** 
    - BFS/DFS graph traversal animation
    - Water Jug Problem solver with step-by-step state visualization
*   **Comparison Tables:** 
    - Search algorithm properties (completeness, optimality, complexity)
    - Inference method comparison (Forward vs. Backward Chaining)
    - Learning approach comparison (supervised vs. unsupervised vs. RL)
*   **Real-World Examples:** 
    - Autonomous vehicles (PEAS framework)
    - GPS navigation (A* algorithm)
    - Medical diagnosis (Backward Chaining)
    - Email spam detection (Supervised Learning)
    - Game AI (Hill Climbing, Reinforcement Learning)
